<memorization_gate_algorithm>
TrainingBleedGate(training_corpus_path: Path, ngram: int = 12):
1. On construction: scan training_corpus_path jsonl; for each row, if row["conversations"][-1]["from"] == "gpt", extract row["conversations"][-1]["value"]; tokenize by str.split() on whitespace; generate sliding 12-grams joined with single space; hash each via xxhash.xxh64_intdigest(gram.encode("utf-8")). Accumulate into self._hashes: set[int].
2. scan(scene_text) -> list[MemorizationHit]:
   - tokens = scene_text.split(); if len(tokens) < ngram: return [].
   - For i in range(len(tokens)-ngram+1): gram = " ".join(tokens[i:i+ngram]); if xxh64_intdigest(gram.encode()) in self._hashes: append MemorizationHit(ngram=gram, position=i).
   - Return list (may be empty).
3. Empty = pass; non-empty = drafter raises ModeADrafterBlocked('training_bleed').

Plan 03-06 CLI constructs gate ONCE per CLI run (preloading 10,751 rows × ~100 12-grams = ~1M ints = ~8MB takes ~5-10s; amortized).

Determinism: xxhash.xxh64_intdigest is stable across processes; no PYTHONHASHSEED risk.
</memorization_gate_algorithm>
