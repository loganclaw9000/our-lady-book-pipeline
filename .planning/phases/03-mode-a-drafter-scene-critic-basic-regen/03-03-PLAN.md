<handshake_flow>
VllmClient.boot_handshake(pin: VoicePinData) -> None:
1. health_ok = GET {base_url}/models with timeout=2s; retry 3x via tenacity. If all retries fail, raise VllmUnavailable("timeout probing /v1/models").
2. Parse JSON; assert any(m["id"] == LORA_MODULE_NAME or m.get("id","").endswith("/paul-voice") for m in data["data"]).
3. actual_sha = compute_adapter_sha(Path(pin.checkpoint_path).expanduser()).
4. if actual_sha != pin.checkpoint_sha: raise VoicePinMismatch(pin.checkpoint_sha, actual_sha, pin.checkpoint_path).
5. Emit one Event via injected EventLogger (optional — CLI wires it):
   - role="vllm_boot_handshake"
   - model="paul-voice"
   - prompt_hash=hash_text(pin.checkpoint_path)
   - input_tokens=0, output_tokens=0
   - latency_ms=<measured>
   - caller_context={module:'drafter.vllm_client', function:'boot_handshake', served_model_id: 'paul-voice', base_url: base_url, vllm_version: (from models response), base_model: pin.base_model}
   - output_hash=actual_sha
   - checkpoint_sha=actual_sha
   - mode="A"
6. Return None (success).

Tenacity retry config: `@tenacity.retry(stop=stop_after_attempt(3), wait=wait_exponential(multiplier=1, min=1, max=4), retry=retry_if_exception_type((httpx.TimeoutException, httpx.ConnectError, httpx.HTTPStatusError)), reraise=True)` — wrapped around `_http_get` and `_http_post` private methods.
</handshake_flow>
