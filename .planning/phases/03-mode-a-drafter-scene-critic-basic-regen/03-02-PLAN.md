<anchor_curation_algorithm>
Selection target: 22 anchors total (midpoint of 20-30 range).
Sub-genre quotas (PITFALLS V-1 warning-sign guard):
- essay: 8 passages (≥6 required)
- analytic: 8 passages (≥6 required)
- narrative: 6 passages (≥4 required)

Source pool:
1. paul-thinkpiece-pipeline/v3_data/train_filtered.jsonl (10,751 clean pairs per memory). Parse each row; extract the `"from":"gpt"` turn's `value`. Apply selection:
   - word count ∈ [150, 400] (tolerance window around the 150-400 target).
   - contains ≥1 em-dash (Paul signature cadence marker) OR ≥1 numeric specificity (regex \b\d+[%$]|\b\d{2,}\b) — voice-presence markers per V-1.
   - sub_genre classification via keyword heuristic:
     - "essay": contains ≥1 of {"writing", "craft", "reading", "what i", "why i", "how i"}.
     - "analytic": contains ≥1 of {"dataset", "benchmark", "token", "model", "evaluation", "score"}.
     - "narrative": contains ≥1 of {"she", "he said", "walked", "turned", "looked"} and dialogue quote mark.
     - First match wins; ties broken by the heuristic order above.
2. Held-out blog posts: tag source_label="blog_heldout". For Phase 3 we tolerate 0 blog rows if the blog corpus is not accessible on disk — anchor_sources.py declares the pointer path(s), and curate_anchors.py gracefully falls back to training-only when the pointer yields 0 rows.

Per-sub-genre selection: sort candidates by a quality proxy (longer text + more em-dashes, tiebreaker=row_index for determinism), take the first N for each sub-genre.

Coverage invariant (tested): compute the norm contribution of each anchor to the centroid. An anchor whose embedding (L2-normalized, dot-product with centroid) exceeds 0.15 is flagged as "dominance risk". Test 4 asserts no flag fires on the real curated set; if the heuristic produces a dominating passage (rare but possible for very on-voice training rows), raise the count to 23/24 and re-run.

Determinism: selection is deterministic given a fixed train_filtered.jsonl (which is committed to paul-thinkpiece-pipeline's head state); running curate-anchors twice produces byte-identical anchor_set_v1.yaml.
</anchor_curation_algorithm>

<anchor_set_sha_algorithm>
anchor_set_sha = SHA256(JSON.dumps(sorted([(a.id, a.text, a.sub_genre) for a in anchors]), sort_keys=True, ensure_ascii=False)).hexdigest()

Fixed encoding: UTF-8, ensure_ascii=False (preserves em-dash + smart quotes).
Field order: (id, text, sub_genre) in that exact tuple order — NOT a dict, so Python tuple ordering is used.
The SHA is stable across machines iff train_filtered.jsonl is byte-identical.
</anchor_set_sha_algorithm>

<scorer_algorithm>
score_voice_fidelity(scene_text, anchor_centroid, embedder) -> float:
1. Validate scene_text is non-empty (raise ValueError on empty).
2. vec = embedder.embed_texts([scene_text])[0]  # shape (1024,) float32
3. vec /= max(np.linalg.norm(vec), 1e-12)  # L2-normalize
4. centroid_norm = max(np.linalg.norm(anchor_centroid), 1e-12)  # defensive
5. cosine = float(np.dot(vec, anchor_centroid) / centroid_norm)
6. Return cosine (in [-1, 1]; typical domain 0.2-0.9).

No caching inside scorer — callers cache at the drafter level if needed. Deterministic given fixed embedder.revision_sha.
</scorer_algorithm>

   ```python
   """score_voice_fidelity — BGE-M3 cosine against the anchor centroid.

   OBS-03: attached to every Mode-A DraftResponse's Event.caller_context.voice_fidelity_score.
   PITFALLS V-1/V-2 band: pass >=0.78, flag 0.75-0.78, fail <0.75, flag >=0.95 (memorization).
   """
   from __future__ import annotations

   from typing import Any

   import numpy as np


   def score_voice_fidelity(
       scene_text: str,
       anchor_centroid: np.ndarray,
       embedder: Any,
   ) -> float:
       if not scene_text or not scene_text.strip():
           raise ValueError("scene_text must be a non-empty string")
       vec = embedder.embed_texts([scene_text])[0].astype(np.float32)

       c_norm = max(float(np.linalg.norm(anchor_centroid)), 1e-12)
       centroid = anchor_centroid / c_norm
       return float(np.dot(vec, centroid))
   ```
