<anthropic_sdk_usage>
STACK.md pins anthropic>=0.96.0 which supports claude-opus-4-7 + client.messages.parse().

Usage pattern:
```python
from anthropic import Anthropic, APIStatusError, APIConnectionError
import tenacity

client = Anthropic()  # reads ANTHROPIC_API_KEY from env

@tenacity.retry(
    stop=tenacity.stop_after_attempt(5),
    wait=tenacity.wait_exponential(multiplier=2, min=2, max=30),
    retry=tenacity.retry_if_exception_type((APIConnectionError, APIStatusError)),
    reraise=True,
)
def _call_opus(messages, system_blocks):
    return client.messages.parse(
        model="claude-opus-4-7",
        max_tokens=4096,
        temperature=0.1,  # low temp for structured judge output
        system=system_blocks,  # list of text blocks with cache_control on the rubric block
        messages=messages,
        response_format=CriticResponse,  # Pydantic schema
    )
```

cache_control for the ephemeral 1h cache — Anthropic 2026-02-05 supports `{"type": "ephemeral", "ttl": "1h"}`:
```python
system_blocks = [
    {
        "type": "text",
        "text": rendered_system_prompt,
        "cache_control": {"type": "ephemeral", "ttl": "1h"},
    },
]
```

User message carries scene_text + context_pack summary; this segment is NOT cached (varies per scene).

messages.parse returns a `.parsed` attribute holding the CriticResponse instance; .content is the raw payload; .usage has token counts including cached_input_tokens.
