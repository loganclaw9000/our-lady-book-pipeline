<conflict_detection_notes>
Per 02-CONTEXT.md, cross-retriever reconciliation extracts structured claims per retriever: `{location, date, possessions, entity_states}`. Phase 2 scope is INTENTIONALLY light — this isn't a full NLI engine.

**W-1 revision — hybrid detection:** Regex-only entity extraction fails on accented / non-Latin-capitalization names common in the Our Lady of Champion corpus:
- `Motecuhzoma` (accented u → `ú` in some variants)
- `Malintzin` (vs romanized "Malinche", "Doña Marina")
- `Tenochtitlan` (vs "Tenochtitlán", "México-Tenochtitlan")
- `Quetzalcoatl` (vs "Quetzalcóatl", "Kukulkán")
- `Cempoala`, `Tlaxcalteca`, etc.

Fix: hybrid detection with an OPTIONAL injected entity_list.
1. For each retrieval hit, `regex_entities = _extract_regex_entities(text)` (original pattern — catches English-style capitalization).
2. If an entity_list is provided: `list_entities = {name for name in entity_list if name in text}` — literal substring match, catches accented forms the regex misses.
3. `entity_candidates = regex_entities | list_entities`.
4. For each entity candidate, extract (location, date, possession) claims via nearby-regex patterns.
5. Diff across retrievers; emit ConflictReport when ≥2 retrievers disagree on same (entity, dimension).

**Kernel discipline:** `conflict_detector.detect_conflicts(retrievals, entity_list=None)` accepts the entity_list as an optional injected parameter. The kernel does NOT import book_specifics. Plan 06 CLI wiring passes `book_specifics.nahuatl_entities.NAHUATL_CANONICAL_NAMES`'s expanded keys+values (flattened to one set) into `ContextPackBundlerImpl(entity_list=...)`.

4. `src/book_pipeline/rag/conflict_detector.py`:
   ```python
   """Cross-retriever conflict detection. PITFALLS R-1 mitigation.

   Extracts structured claims (location/date/possession) from each retrieval hit
   and diffs across retrievers. Emits ConflictReports when >= 2 retrievers disagree
   on the same (entity, dimension) pair.

   W-1 revision: regex-only entity detection MISSES accented/non-Latin-capitalization
   names common in the Mesoamerican corpus (Motecuhzoma, Malintzin, Tenochtitlán).
   Hybrid approach: entity_candidates = union(regex_matches, entity_list_hits). The
   entity_list is INJECTED by the CLI composition layer (Plan 06 wires in
   book_specifics.nahuatl_entities). Kernel stays clean — no book_specifics import.

   Scope is deliberately SIMPLE for Phase 2 — this is a FORCING FUNCTION, not an NLI engine.
   Thesis 005 in Phase 6 may replace this with a more sophisticated detector.
   """
   from __future__ import annotations
   import re
   from book_pipeline.interfaces.types import ConflictReport, RetrievalResult

   # Regex patterns for English-style capitalization — complement the injected entity_list
   _LOCATION_PHRASE_RE = re.compile(
       r"(?:is at|arrives at|stays at|returns to|in)\s+([A-Z][\w\s\-]+?)\b",
   )
   _POSSESSION_PHRASE_RE = re.compile(
       r"(?:has|carries|possesses|holds)\s+(?:the\s+)?(\w+)\b",
   )
   _DATE_RE = re.compile(r"\b(\d{4}(?:-\d{2}-\d{2})?)\b")
   _NAME_REGEX_RE = re.compile(r"\b([A-Z][a-z]+(?:\s+[A-Z][a-z]+)?)\b")

   def _extract_entity_candidates(text: str, entity_list: set[str] | None) -> set[str]:
       """W-1: hybrid — regex matches ∪ entity_list substring hits.

       entity_list supplies non-Latin/accented canonical forms (Motecuhzoma, Malintzin, ...)
       that English-capitalization regex would miss.
       """
       regex_hits = {m.group(1).strip() for m in _NAME_REGEX_RE.finditer(text)}
       list_hits: set[str] = set()
       if entity_list:
           for name in entity_list:
               if name and name in text:
                   list_hits.add(name)
       return regex_hits | list_hits

   def _extract_claims_for_entity(text: str, entity: str) -> list[tuple[str, str]]:
       """For a given entity, scan text for (dimension, value) claims."""
       out: list[tuple[str, str]] = []
       # Look for the entity mention and its surrounding context (window of ~80 chars).
       for m in re.finditer(re.escape(entity), text):
           window = text[max(0, m.start() - 80): m.end() + 80]
           for lm in _LOCATION_PHRASE_RE.finditer(window):
               out.append(("location", lm.group(1).strip()))
           for pm in _POSSESSION_PHRASE_RE.finditer(window):
               out.append(("possession", pm.group(1).strip()))
           for dm in _DATE_RE.finditer(window):
               out.append(("date", dm.group(1).strip()))
       return out

   def detect_conflicts(
       retrievals: dict[str, RetrievalResult],
       entity_list: set[str] | None = None,
   ) -> list[ConflictReport]:
       """Diff (entity, dimension) claims across retrievers; emit a report per disagreement.

       Args:
           retrievals: dict keyed by retriever name -> RetrievalResult.
           entity_list: W-1 — optional set of canonical entity names the caller wants to
               detect (e.g., Mesoamerican names from book_specifics.nahuatl_entities).
               When None, falls back to regex-only entity extraction (backwards-compat).

       Returns deterministic list of ConflictReport (sorted by entity, dimension).
       """
       # (entity, dimension) -> retriever_name -> set(values)
       claims_by_key: dict[tuple[str, str], dict[str, set[str]]] = {}
       evidence: dict[tuple[str, str], dict[str, list[str]]] = {}
       for retriever_name, rr in retrievals.items():
           for hit in rr.hits:
               entities = _extract_entity_candidates(hit.text, entity_list)
               for entity in entities:
                   for dim, value in _extract_claims_for_entity(hit.text, entity):
                       key = (entity, dim)
                       claims_by_key.setdefault(key, {}).setdefault(retriever_name, set()).add(value)
                       evidence.setdefault(key, {}).setdefault(retriever_name, []).append(hit.chunk_id)
       out: list[ConflictReport] = []
       for (entity, dim), by_ret in sorted(claims_by_key.items()):
           collapsed = {r: "|".join(sorted(vals)) for r, vals in by_ret.items() if vals}
           if len(set(collapsed.values())) >= 2:  # at least 2 retrievers disagree
               out.append(ConflictReport(
                   entity=entity,
                   dimension=dim,
                   values_by_retriever=collapsed,
                   source_chunk_ids_by_retriever=evidence[(entity, dim)],
                   severity="mid",
               ))
       return out
   ```
