<golden_queries_format>
tests/rag/golden_queries.jsonl — each line is a JSON object:

```json
{
  "query_id": "historical_01",
  "axis": "historical",
  "scene_request": {
    "chapter": 3,
    "scene_index": 2,
    "pov": "Andrés",
    "date_iso": "1519-08-16",
    "location": "Cempoala",
    "beat_function": "arrival",
    "preceding_scene_summary": null
  },
  "expected_chunks": [
    {"source_file_suffix": "our-lady-of-champion-brief.md", "heading_path_substr": "Historical > Cempoala"}
  ],
  "forbidden_chunks": [
    {"source_file_suffix": "our-lady-of-champion-engineering.md", "heading_path_substr": "Hypotheticals"}
  ]
}
```

- `axis` identifies which retriever's top-8 we check against (the historical retriever's RetrievalResult).
- `expected_chunks`: a list of {source_file_suffix, heading_path_substr}; MATCH is: every expected_chunk has AT LEAST ONE hit in that axis's top-8 where `hit.source_path.endswith(suffix)` AND `substr in hit.metadata["heading_path"]`.
- `forbidden_chunks`: any hit in ANY retriever (not just the axis) matching (suffix + substr) is a FAIL.

Target: ≥12 queries total with ≥2 per axis (10 axis-specific + 2 cross-checks). Pass criteria per 02-CONTEXT.md:
- ≥90% of queries have ALL expected_chunks matched in top-8
- 0 forbidden_chunks appear in any retriever for any query

Seed queries drawn from outline.md beats — Cholula arrival (ch10), two-thirds revelation (ch17-18), siege opening (ch25), plus routine beats (ch1, ch3, ch7). Each axis should have at least one "happy path" query and one "anti-leak" query (where the right chunk should come from axis X, and axis Y MUST NOT surface it).
</golden_queries_format>

<expected_chunks_snapshot_notes>
To pin against the Plan 02 baseline, Task 1 captures a snapshot of (source_file, heading_path) tuples from the actually-populated LanceDB tables at the moment of the baseline ingest. This is written to tests/rag/fixtures/expected_chunks.jsonl and committed. If a future ingest has a different chunker revision, the snapshot won't match → golden-query test fails → human inspects whether the chunker change is intentional.

The snapshot is NOT the golden_queries.jsonl expectation set (those are authored manually). The snapshot is a PROBE the test uses to detect "did anything land in the index that matches this suffix+substr at all" — an absent-expected-chunk case is diagnosed as "chunk no longer exists in the index" vs "chunk exists but didn't rank top-8".
</expected_chunks_snapshot_notes>

3. `tests/rag/test_golden_queries.py`:
   ```python
   import json
   from collections import Counter
   from pathlib import Path
   import pytest
   from book_pipeline.interfaces.types import SceneRequest
   from book_pipeline.rag import ContextPackBundlerImpl
   # ... construct retrievers from config/rag_retrievers.yaml + real indexes/

   GOLDEN_PATH = Path(__file__).parent / "golden_queries.jsonl"
   INDEXES_DIR = Path(__file__).parent.parent.parent / "indexes"

   def _load_queries():
       return [json.loads(line) for line in GOLDEN_PATH.read_text().splitlines() if line.strip()]

   def test_golden_queries_jsonl_schema():
       for q in _load_queries():
           assert set(q) >= {"query_id","axis","scene_request","expected_chunks","forbidden_chunks"}
           SceneRequest.model_validate(q["scene_request"])

   def test_golden_queries_coverage():
       axes = Counter(q["axis"] for q in _load_queries())
       for axis in ("historical","metaphysics","entity_state","arc_position","negative_constraint"):
           assert axes[axis] >= 2, f"axis {axis} has only {axes[axis]} queries; need >=2"

   @pytest.mark.skipif(not INDEXES_DIR.exists() or not any(INDEXES_DIR.iterdir()), reason="indexes/ is empty; run `book-pipeline ingest --force` first")
   @pytest.mark.slow
   def test_golden_queries_pass_on_baseline_ingest():
       # Construct real bundler + real retrievers
       queries = _load_queries()
       results = []  # list of (query_id, expected_ok: bool, forbidden_leaks: list)
       for q in queries:
           req = SceneRequest.model_validate(q["scene_request"])
           pack = bundler.bundle(req, retrievers)
           axis_hits = pack.retrievals[q["axis"]].hits
           expected_ok = all(
               any(h.source_path.endswith(e["source_file_suffix"]) and e["heading_path_substr"] in h.metadata.get("heading_path","")
                   for h in axis_hits)
               for e in q["expected_chunks"]
           )
           forbidden_leaks = []
           for axis_name, rr in pack.retrievals.items():
               for h in rr.hits:
                   for f in q["forbidden_chunks"]:
                       if h.source_path.endswith(f["source_file_suffix"]) and f["heading_path_substr"] in h.metadata.get("heading_path",""):
                           forbidden_leaks.append((axis_name, f, h.chunk_id))
           results.append((q["query_id"], expected_ok, forbidden_leaks))
       num_pass_expected = sum(1 for (_, ok, _) in results if ok)
       total_leaks = sum(len(leaks) for (_, _, leaks) in results)
       pct = num_pass_expected / len(queries)
       assert total_leaks == 0, f"Forbidden leaks: {[(qid, leaks) for (qid, _, leaks) in results if leaks]}"
       assert pct >= 0.90, f"Only {pct:.0%} of queries returned their expected chunks in top-8"
   ```
